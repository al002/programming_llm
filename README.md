## Getting started

### Install dependencies
```
poetry install
```

Main dependencies

- [LlamaIndex](https://github.com/jerryjliu/gpt_index)
- [LangChain](https://github.com/hwchase17/langchain)

### Construct index first

```
python construct_index.py
```

### Start gRPC server

```
python llm_server.py
```
